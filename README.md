# Turnero SaaS

Software orientado a distintos profesionales y comercios para que puedan **gestionar sus turnos** y el seguimiento de sus **pacientes o clientes** de forma sencilla y práctica.

Está pensado para rubros como estética, barberías, psicología, nutrición, medicina y otros servicios profesionales. Cada negocio administra su agenda, servicios y equipo; los clientes pueden reservar turnos online y recibir recordatorios automáticos.

## ¿Qué permite hacer?

- **Profesionales y comercios:** configurar horarios, servicios, agenda semanal, fichas de seguimiento y caja.
- **Clientes:** registrarse una vez en la plataforma y reservar turnos en cualquier comercio adherido.
- **Automatización:** recordatorios y avisos por WhatsApp y correo electrónico.

## Estructura del proyecto

Monorepo con tres carpetas principales:

| Carpeta | Descripción |
|---------|-------------|
| `front/` | Aplicación web (Angular + Bootstrap) |
| `backend/` | API REST (Node.js + Express + TypeScript) |
| `database/` | Diagrama ERD de base de datos |
| `backend/prisma/` | Esquema Prisma y migraciones |
